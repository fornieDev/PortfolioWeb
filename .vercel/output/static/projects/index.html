My Physio

My Physio is a Physiotherapy ERP whose first module is a social network that allows patients and physiotherapists to communicate to send custom exercises routines to make at home.

Technologies: developed for Android with Android Studio in Kotlin,with Firebase Data Base. Involvement: unipersonal project. Roles: design, fullstack development and testing.



Server socket chatbot

Server socket chatbot is a school project of Services and process programming course,and it's a chat bot which communicates the client and the server through the console,and the goal is to return a password based on the parameters generated by the server.

Technologies: developed in Eclipse with Java. Involvement: unipersonal project. Roles: design, development and testing.



Zoominoutron

Zoominoutron is a puzzle platform videogame developed for PC made with Unity where we control Nikola, a science prodigy kid who after an accident, gets the ability to make him bigger or smaller at will.We must use this power to escape from the room where we got stuck after the accident to find a cure.

Technologies: developed in Unity and Visual Studio with C#. For the save game it uses a json local file. Involvement: unipersonal project. Roles: design, fullstack development, testing.



Unity Essentials Project

Technologies: developed in Unity and Visual Studio with C#. Involvement: unipersonal project. Roles: fullstack developement.



Cube Clicker

Cube Clicker is an Android game with a cube that has faces colored differently, and by clicking the cube it boosts upward.We must avoid the cube from falling down,with the additional difficulty that every click changes the background to the same color of the face,and if we click a face of the color of the background the game ends.

Technologies: developed in Unity and Visual Studio with C#. Involvement: unipersonal project. Roles: design,developement and testing.
